## ðŸŽ“ Learning Resources

### Understanding Priority Scores:
```
Priority Score = weighted sum of:
- Severity (0-25 points)
- Impact (0-25 points)
- Urgency (0-25 points)
- Complexity (0-25 points)
```

### Health Score Calculation:
```
Health Score = 100 - (
  critical_count * 20 +
  high_priority_count * 10 +
  medium_priority_count * 5 +
  low_priority_count * 2
)
```

### Timeline Assignment:
- **Immediate:** Priority score > 80 OR severity = critical
- **This Week:** Priority score 60-80
- **This Month:** Priority score 40-60
- **Next Quarter:** Priority score < 40
